# Head and Body

## HTML template

There is a set of elements that make make up the basic structure of every HTML document. The following code snippet shows the prototypical structure.

```markup
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>My first page</title>
  </head>
  <body>
    <p>A paragraph with text.</p>
  </body>
</html>
```
