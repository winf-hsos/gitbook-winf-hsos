# HTML

## Das Grundgerüst

Jede Seite beinhaltet eine Menge an Elementen, die auch das HTML-Grundgerüst genannt werden:

```markup
<html>
    <head>
        <title>Titel der Webseite</title>
    </head>
   
    <body>
        <h1>Das ist eine Überschrift</h1>
        <p>Das ist ein Absatz</p>    
    </body>
</html>
```
